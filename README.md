# Automation Utility
A Python-based tool built to automate repetitive desktop workflows.

## Features
- Automates screen interactions using PyAutoGUI.
- Designed to minimize manual input for data-heavy tasks.

## Why I built this
As a student balancing academic research and e-commerce logistics in Oran, I created this tool to reduce time spent on repetitive tasks, allowing me to focus on higher-level problem solving and project scaling.
